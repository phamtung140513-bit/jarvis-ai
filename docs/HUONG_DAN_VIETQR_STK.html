Hướng dẫn VietQR + STK của bạn

Video demo chỉ là ví dụ luồng — không dùng STK trong video. Hệ thống gắn tài khoản ngân hàng trong file .env của shop.

1. STK nhận tiền (máy bạn)

Ngân hàngHDBank (hdbank)
Số TK0964084203
Chủ TKNGUYEN THI THUY

Sửa ở 2 file (phải giống nhau):

  • C:\Users\Admin\vietqr-pay\.env — BANK_CODE / BANK_ACCOUNT / BANK_ACCOUNT_NAME
  • C:\Users\Admin\Jarvis-AI\.env — BANK_ID / BANK_ACCOUNT / BANK_ACCOUNT_NAME

2. Ba mode VietQR

Mode Cần gì Ghi chú
image Chỉ STK (đang bật) QR qua img.vietqr.io + STK bạn — không cần API key
io Client ID + API Key từ my.vietqr.io API api.vietqr.io/v2/generate + STK bạn
partner Token partner + URL HTTPS Biết đã CK (webhook BĐSD) → auto lên gói

3. Bật API chính thức (mode io)

  1. Đăng ký / đăng nhập my.vietqr.io
  2. Lấy Client ID và API Key
  3. Sửa vietqr-pay\.env:
VIETQR_MODE=io
VIETQR_CLIENT_ID=...dán key...
VIETQR_API_KEY=...dán key...
BANK_CODE=hdbank
BANK_ACCOUNT=0964084203
BANK_ACCOUNT_NAME=NGUYEN THI THUY

Rồi restart: cd C:\Users\Admin\vietqr-pay → npm start

4. Chạy dịch vụ

# Terminal 1 — VietQR (STK bạn)
cd C:\Users\Admin\vietqr-pay
npm start

# Terminal 2 — Web AI
cd C:\Users\Admin\Jarvis-AI
.\.venv\Scripts\python.exe -m webapp.server

# Terminal 3 — Bot Telegram (fulfill)
cd C:\Users\Admin\Jarvis-AI
.\.venv\Scripts\python.exe launcher.py

Kiểm tra STK: http://127.0.0.1:3000/api/health → phải thấy bank=hdbank, accountMasked=096***203.
